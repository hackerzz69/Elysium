# Elysium RSPS Website Starter

Static icy-glassmorphism starter site for an unofficial RSPS MMORPG.

## Files
- `index.html` - homepage
- `css/style.css` - full responsive theme
- `js/main.js` - mobile nav, reveal animations, mock player count, snow particles
- `assets/elysium-logo.png` - transparent full logo
- `assets/elysium-icon.png` - transparent icon

## Quick hosting
Upload the contents of this folder to GitHub Pages, Netlify, Vercel, or any web host.
